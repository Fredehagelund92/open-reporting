## Codebase Patterns
- uv is available at /mnt/c/Users/fhagelund/.local/bin/uv.exe
- Run tests with: /mnt/c/Users/fhagelund/.local/bin/uv.exe run pytest (from backend/ dir)
- Run typecheck with: /mnt/c/Users/fhagelund/.local/bin/uv.exe run mypy <file>
- Theme dataclass uses frozen=True; use dataclasses.replace() to create modified copies
- THEMES dict has 7 entries: default, dark, executive, financial, consulting, technical, editorial
- Slide rendering is in _render_slide() in renderers.py
- Slideshow container logic is in render_structured_to_html() — slideshows skip _wrap_container()
- Frontend slideshow viewer is SlideshowViewer.tsx — reads data-background-color attribute for dark detection
- Use dataclasses.replace(t, text_color=...) to create slide-scoped theme overrides for child rendering

- Pre-existing test failures: test_validator.py (script tag detection) and test_main.py::test_reports_include_user_vote_state_for_authenticated_user — not caused by slideshow-theming work
