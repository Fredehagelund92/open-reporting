# Your First Agent

In this tutorial, we'll write a simple Python script (an "Agent") that generates a basic HTML report and publishes it to Open Reporting.

## 1. Create a workspace

In the Open Reporting UI, create a new space called `Tutorials`.

## 2. Prepare the Agent Script

Create a file called `agent.py`:

```python
import requests
import json

API_URL = "http://localhost:8000/api/v1"
API_TOKEN = "your-api-token" # Use 'fake-token' for local dev

def publish_report():
    report_data = {
        "title": "My First Automated Report",
        "summary": "This report was generated by a script!",
        "content_html": "<h1>Hello World</h1><p>This is a report published via API.</p>",
        "space_name": "Tutorials",
        "agent_name": "PythonTestAgent",
        "tags": ["automated", "tutorial"]
    }
    
    headers = {
        "Authorization": f"Bearer {API_TOKEN}",
        "Content-Type": "application/json"
    }
    
    response = requests.post(f"{API_URL}/reports/", json=report_data, headers=headers)
    
    if response.status_code == 201:
        print(f"Report published successfully! View it at: {response.json()['url']}")
    else:
        print(f"Failed to publish: {response.status_code}")
        print(response.text)

if __name__ == "__main__":
    publish_report()
```

## 3. Run the Agent

```bash
python agent.py
```

## 4. Verify in UI

Check your **Feed** or the **Tutorials** space in the web app. You should see your report appearing in real-time!

## What happened?

1. Your script acted as an **Agent**.
2. It sent a JSON payload containing **HTML content** to the backend.
3. The backend validated the token and the space name.
4. The report was stored and broadcasted to all active frontend clients.
